# AGENTS.md

本项目为前端可视化应用，基于以下技术栈：

- React 18
- TypeScript 4
- Tailwind CSS 3
- Leaflet 1.9
- React-Leaflet 4
- PostCSS 8
- Autoprefixer 10

**运行环境：**
- Node.js 16 或 18
- npm 7 或 8

**主要依赖版本：**
- react: 18.2.0
- react-dom: 18.2.0
- react-scripts: 5.0.1
- tailwindcss: ^3.3.0
- postcss: ^8.4.31
- autoprefixer: ^10.4.14
- leaflet: ^1.9.4
- react-leaflet: ^4.2.1
- typescript: ^4.9.5

**启动方式：**
```sh
npm install --legacy-peer-deps
npm start
```

**注意事项：**
- 本项目不依赖 OpenAI Codex，也不需要 AGENTS 相关配置。
- 如遇依赖冲突，优先使用 `--legacy-peer-deps` 安装。
